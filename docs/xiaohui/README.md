## 🎯 DevLoop 与小辉的关系

```
DevLoop (独立开源产品)
    │
    │  从小辉剥离
    │
    ▼
小辉数字员工 2.0 (商业产品)
    │
    │  持续集成 DevLoop 调度能力
    │
    ▼
企业客户 (飞书/企微/微信用户)
```

**DevLoop 拿走的**: 数据底座 (任务总线/经验库/铁律) + 智能体调度 (8 角色)
**DevLoop 留的**: 0 业务耦合 — 不包含 CRM/ERP/商城等业务模块
