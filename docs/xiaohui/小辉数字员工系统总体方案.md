# 小辉数字员工 2.0 — 系统总体方案

> 版本: V1.1 | 日期: 2026-05-15 | DevLog: 602任务/590+已完成

## 一、系统总览

小辉数字员工 2.0 是基于 OpenClaw 框架构建的 AI 驱动企业数字员工系统。

**技术栈**: Node.js + Fastify 4 + Prisma 5 + PostgreSQL 15 + Redis 7 / Vue 3 + Element Plus + Vite
**AI 层**: OpenClaw gateway + MiniMax/千问 API
**通道**: 飞书 / 企业微信 / 钉钉

```
六大核心板块:
┌──────────┬──────────┬──────────┬──────────┬──────────┬──────────┐
│ 认证权限 │ ERP核心  │ 智能商城 │ 售后维修 │ 系统配置 │ 智能工作伴侣│
│ 10路由   │ 21模块   │ 12+24   │ 2路由    │ 5路由    │ 3模块    │
│ 2中间件  │ 40+路由  │ 27张表  │          │          │ 14技能   │
└──────────┴──────────┴──────────┴──────────┴──────────┴──────────┘
                      OpenClaw AI 网关 + 多消息通道
```

## 七、板块六：智能工作伴侣 (60%)

这是系统的 AI 大脑，负责理解用户意图、路由消息、调用工具、执行任务。
用户通过飞书/企微/钉钉发消息 → Agent 理解意图 → 调用后端 API → 返回结果。

### 7.1 核心架构

```
用户消息(飞书/企微/钉钉)
      ↓
gateway.js          ← 消息入口（公开端点，IM webhook回调）
      ↓
openclaw.js         ← OpenClaw深度集成（意图识别+工具执行+上下文）
      ↓
intent-recognizer   ← 关键词匹配(15条粗粒度规则)
intent-classifier   ← 精确意图分类
qianwen-intent      ← 千问AI意图识别（语义理解）
hybrid-recognizer   ← 混合识别（规则+AI）
      ↓
intent-handler      ← 意图处理分发
gateway-service     ← 业务路由到具体模块
      ↓
OpenClaw技能系统    ← 14个技能（打印/商城购物/Excel/PDF/开发等）
      ↓
后端API             ← 执行具体业务操作
      ↓
返回自然语言回复
```

### 7.2 Agent 实例管理 (agent模块)
- agent-instances.js: Agent实例CRUD（创建/配置/启停）
- agent.js: Agent核心（模型选择/参数配置）
- agent-tasks.js: Agent任务队列（异步任务调度）

### 7.3 AI 陪伴与学习 (companion模块)
- routes/: 陪伴对话接口
- scheduler.js: 学习调度器（每日3:00 + 每周一4:00自动学习）
