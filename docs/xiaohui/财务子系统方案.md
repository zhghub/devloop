# 财务子系统方案

> 版本: V1.0 | 日期: 2026-05-15
> 父文档: docs/小辉数字员工系统总体方案.md 板块二

## 一、定位
企业财务数据中枢，覆盖应收应付、发票管理、财务报表、成本核算。
后端7个路由文件，前端7个页面，路径前缀 `/api/v1/erp/*`

## 二、路由清单
finance.js(应收应付+付款申请) / invoices.js(发票CRUD+红字) / invoice-inventory.js(号段) / payments.js(付款记录) / financial-reports.js(3大报表) / cost-accounting.js(BOM成本) / payment-gateway.js(付款网关)

## 三、完成度
应收应付 90% | 发票管理 85% | 发票库存 80% | 财务报表 75% | 成本核算 70% | 税控自动开票 0% | 发票OCR 0%

## 四、认证模式
GET用preHandler:requireAuth(), POST/PUT内部调verifySuperAdmin(permission>=100)。与quote模块一致的人工JWT模式。
